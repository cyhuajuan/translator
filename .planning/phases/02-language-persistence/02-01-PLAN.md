  <action>
    Modify src/routes/index.tsx to add persistence for sourceLang and targetLang.

    REQUIRED CHANGES:

    1. Add store import at top:
       import store from '@/lib/store'

    2. Add new state for store loading:
       const [storeLoaded, setStoreLoaded] = React.useState(false)

    3. Add useEffect to load persisted values AFTER existing state declarations (around line 21):
       ```typescript
       // Load persisted language preferences on mount
       React.useEffect(() => {
         const loadPrefs = async () => {
           try {
             await store.init()
             const savedSource = await store.get<string>('sourceLang')
             const savedTarget = await store.get<string>('targetLang')
             if (savedSource) setSourceLang(savedSource)
             if (savedTarget) setTargetLang(savedTarget)
           } catch (e) {
             // Silently fall back to defaults (en->zh) per research recommendation
           }
           setStoreLoaded(true)
         }
         loadPrefs()
       }, [])
       ```

    4. Modify the Select onChange handlers to save to store:

       Change line 83 from:
         onChange={(e) => setSourceLang(e.target.value)}
       To:
         onChange={async (e) => {
           setSourceLang(e.target.value)
           await store.set('sourceLang', e.target.value)
         }}

       Change line 98 from:
         onChange={(e) => setTargetLang(e.target.value)}
       To:
         onChange={async (e) => {
           setTargetLang(e.target.value)
           await store.set('targetLang', e.target.value)
         }}

    IMPORTANT: The async onChange handlers must NOT await store.init() - init() was already called in useEffect. If store is called before init, it will fail silently.

    Default values remain: sourceLang="en", targetLang="zh" (per D-03)
  </action>
