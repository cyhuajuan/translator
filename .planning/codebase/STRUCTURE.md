## Directory Purposes

**`src/`:**
- Purpose: React frontend source code
- Contains: Routes, components, utilities, styles

**`src/routes/`:**
- Purpose: File-based routing via TanStack Router
- Contains: Route components (e.g., `index.tsx`, `__root.tsx`)
- TanStack Router plugin generates `routeTree.gen.ts` from these files

**`src/components/ui/`:**
- Purpose: shadcn/ui component library
- Contains: Reusable UI components (e.g., `button.tsx`)
- Styled with Tailwind CSS

**`src/lib/`:**
- Purpose: Shared utilities
- Contains: Utility functions like `cn()` for Tailwind class merging

**`src/styles/`:**
- Purpose: Global CSS styles
- Contains: Tailwind imports and global styles

**`src-tauri/src/`:**
- Purpose: Rust backend source
- Contains: `lib.rs` (app builder), `main.rs` (entry)

**`src-tauri/capabilities/`:**
- Purpose: Tauri 2 capability permissions
- Contains: `default.json` for default permissions

## Key File Locations

**Entry Points:**
- `src/main.tsx`: React app entry, router and QueryClient setup
- `index.html`: HTML entry point, mounts to `#root`
- `src-tauri/src/main.rs`: Rust entry, calls `hj_project_name_lib::run()`
- `src-tauri/src/lib.rs`: Tauri app builder with command handlers

**Configuration:**
- `vite.config.ts`: Vite bundler config, path aliases (`@/` -> `./src/`)
- `tsconfig.json`: TypeScript strict mode, path aliases
- `biome.json`: Linting and formatting rules
- `src-tauri/tauri.conf.json`: Tauri app metadata, window config

**Generated Files:**
- `src/routeTree.gen.ts`: Auto-generated by TanStack Router plugin (should not edit)
- `src-tauri/target/`: Rust build output (generated)

## Naming Conventions

**Files:**
- Route files: `camelCase.tsx` or `kebab-case.tsx` based on route path
- Components: `PascalCase.tsx`
- Utilities: `camelCase.ts`
- Config: `camelCase.json` or `kebab-case.json`

**Functions:**
- Rust: `snake_case` (e.g., `greet`, `generate_handler`)
- TypeScript: `camelCase`

**Types/Classes:**
- TypeScript: `PascalCase`
- Rust: `PascalCase`

## Where to Add New Code

**New Route:**
- Primary: Create `src/routes/[route-name].tsx`
- Uses `createFileRoute('/route-name')` pattern
- TanStack Router auto-generates types in `routeTree.gen.ts`

**New UI Component:**
- Implementation: Add to `src/components/ui/[component-name].tsx`
- Follow shadcn/ui patterns with `class-variance-authority`
- Use `cn()` utility for Tailwind class merging

**New Utility:**
- Shared helpers: `src/lib/utils.ts`
- If utils grow large, create new files like `src/lib/api.ts`

**New Tauri Command:**
- Add to `src-tauri/src/lib.rs` with `#[tauri::command]`
- Invoke from frontend via `@tauri-apps/api`

**New Rust Module:**
- Add to `src-tauri/src/lib.rs` or create submodules
- Export via `pub fn`

## Special Directories

**`src-tauri/target/`:**
- Purpose: Rust compilation output
- Generated: Yes
- Committed: No (typically in `.gitignore`)

**`node_modules/`:**
- Purpose: npm/pnpm dependencies
- Generated: Yes
- Committed: No

**`dist/`:**
- Purpose: Production build output
- Generated: Yes
- Committed: No

**`.planning/codebase/`:**
- Purpose: GSD architecture/planning documents
- Generated: No
- Committed: Yes
